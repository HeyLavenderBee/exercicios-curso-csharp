# Listas de Exercícios Curso C#
Conjunto de todas as listas de exercícios do curso de C#, para treinar a sintaxe com a linguagem, assim como a programação orientada a objetos. <br>
Todo o conteúdo está separado em capítulos, que começa a partir do Capítulo 3. E dentro de cada capítulo, as listas são separadas por seus respectivos nomes, contendo um projeto para cada listas, com todos os códigos dentro de arquivos Program.cs.
